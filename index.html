Outlier detector

Introduction

This is a simple tool that lets you upload an arbitrary dataset and finds any outliers. This can be useful as a starting point for finding data which is out of the ordinary and might be worthy of further investigation.

Upload your data

Your file should be comma separated. Each row is an observation and each column is a variable. The first column of each row should contain a label for that observation. The tool finds observations that are outliers. Here are two examples:

  • Municipal wage costs as a percentage of operating revenue
  • Municipal fine revenue as a percentage of operating revenue
  • Youth attendance at school by age.

Explanation

This tool uses Principal Component Analysis (PCA) to reduce the dimensionality of the data. In other words, it reduces the number of variables in an observation to two. It does this while attempting to retain the variance in the data. Reducing the number of variables allows us to plot each observation in two dimensions and makes it easier to identify outliers.
